¿Qué es SEO?

SEO (Search Engine Optimization) se refiere a la forma en la que posicionamos nuestra página web, procurando encontrar la forma más eficiente de hacerlo.

Principales beneficios de usar SEO

Su principal beneficio es que una web bien posicionada y adecuadamente configurada tendrá un alcance mayor. Esto facilitará las visitas de futuros usuarios y hará que aparecer en los primeros resultados de los motores de búsqueda sea una tarea más fácil.

¿Cómo podemos optimizar nuestra página web?

En los últimos años el mercado de dispositivos móviles ha crecido enormemente, al igual que lo han hecho las aplicaciones que se alojan en la nube. Por lo que en la época actual es de suma importancia diseñar sitios web "responsive" que se adapten a las resoluciones de diferentes dispositivos. Aunque también es importante considerar las meta etiquetas.

Anteriormente los motores de búsqueda priorizaban las meta etiquetas para el posicionamiento de un sitio web. Aunque hoy en día no sea la prioridad número uno, sigue siendo muy importante entender el funcionamiento de estas etiquetas y cómo pueden aportar al posicionamiento de nuestra página web.

Meta Etiquetas

Las meta etiquetas se diferencian del resto de etiquetas HTML porque permiten al motor de búsqueda conocer detalles específicos del sitio web. Entre ellos destacan el título, la descripción de la página, palabras clave y el autor del sitio web.

Cada una de estas etiquetas se colocan en la etiqueta head de la página.

Title

Esta etiqueta se encarga de determinar el título de la página.

<title>Posicionamiento de páginas web (SEO)</title>

Charset

Esta etiqueta permite elegir la codificación de carácteres que utilizará la página. En idiomas como el español es muy importante elegir una codificación adecuada para reflejar letras o símbolos especiales del idioma.

<meta charset="UTF-8">

Description

Esta etiqueta permite dar una corta descripción de la página. Es importante cuidar la ortografía y lo largo pues es lo que se reflejará en la búsqueda.

<name="description" content="El SEO se refiere a las formas para posicionar una página web eficientemente. ¡Aprende más aquí!"/>

Keywords

Con las palabras clave podemos definir algunas palabras importantes o que destacan el mensaje de nuestra web.

<meta name="keywords" content="spain, españa, seo"/>

Author

Permite indicar el autor del sitio web.

<meta name="author" content="Sebastián Ulloa Campos">

Resumen

Es importante implementar el uso de las meta etiquetas para asegurar un mejor posicionamiento de la página, y con ello alcanzar a más futuros visitantes.
